⚡ VERIFY YOUR SETUP RIGHT NOW

Run this to check if everything is configured:

    npm run verify-env

You'll see:
  ✅ = Variable is set correctly
  ⚠️  = Variable has placeholder (needs update)
  ❌ = Variable is missing

═══════════════════════════════════════════════════════════════════════════════

🎯 IF YOU'RE GETTING FIREBASE ERRORS

1. Run: npm run verify-env
2. Look for ❌ or ⚠️  symbols
3. Update your .env.local with missing credentials
4. Restart: npm run dev

When deploying to Vercel after Firebase account change:

[ ] Test locally: npm run dev
[ ] Update Vercel Settings > Environment Variables
[ ] Redeploy
[ ] Test on Vercel domain
